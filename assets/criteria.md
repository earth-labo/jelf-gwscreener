# ClimateWash診断基準リスト（EU指令ベース）- 改訂版

## 基本情報
- **対象指令**：
  - EU消費者エンパワメント指令（Directive 2024/825） ✓ 施行済
  - EUグリーンクレーム指令（提案版 COM(2023) 166） ※2025年6月撤回表明、ただし規制内容は参照価値あり
- **診断対象**：Climate Wash（気候関連のグリーンウォッシュ）
- **適用開始**：2026年9月27日（EU消費者エンパワメント指令）
- **評価レベル**：High / Medium / Low + 点数化（100点満点）

## 1. EU消費者エンパワメント指令による禁止事項（Per Se Prohibitions）

これらは「常に不公正」（Annex I）とされる行為で、個別判断不要で違反となります。

### 1.1 一般的・抽象的な環境主張の禁止（Generic Environmental Claims）
**法的根拠**: Directive 2024/825, Annex I, Point 11b

**禁止される表現例（原文）：**
- "climate-neutral" / "climate-friendly" / "carbon friendly"
- "CO2 neutral" / "carbon neutral" / "CO2 compensated"
- "net zero" / "climate net zero"
- "eco-friendly" / "eco" / "green"
- "nature's friend" / "ecological" / "environmentally correct"
- "energy efficient"
- "biodegradable" / "biobased"
- "gentle on the environment"

**日本語で該当する表現：**
- 「気候中立」「気候に優しい」「カーボンフレンドリー」
- 「カーボンニュートラル」「CO2ニュートラル」
- 「CO2ゼロ」「実質ゼロ」「排出ゼロ」
- 「ネットゼロ」「正味ゼロ」
- 「脱炭素」「ゼロカーボン」
- 「エコ」「エコフレンドリー」
- 「環境に優しい」「地球に優しい」
- 「グリーン」「自然派」「エコロジカル」
- 「省エネ」「エネルギー効率的」
- 「生分解性」「バイオベース」
- 「環境配慮」「環境対応」「環境負荷低減」

**重要な例外規定:**
以下の場合、一般的表現でも使用可能：
1. **EUエコラベル**（Regulation 66/2010）認証済み
2. **ISO 14024 Type I ecolabel**（加盟国公認のもの）
3. **EU法に基づく卓越した環境性能**（例：エネルギーラベルClass A）
4. **他のEU法令で定められた最高レベルの環境性能**

さらに、**同じ媒体上で明確に具体化されている場合**は一般的表現ではない：
- ❌「気候に優しいパッケージ」→ 一般的表現（禁止）
- ✅「このパッケージの製造に使用したエネルギーの100%は再生可能エネルギー由来」→ 具体的表現（OK）

**色彩・画像との組み合わせ:**
文字・口頭表現が色彩や画像と組み合わされた場合も、一般的環境主張とみなされる可能性がある。

**診断ポイント（High risk）：**
- 上記の一般的表現を卓越した環境性能の証明なしに使用 → High（20-30点減点）
- 具体的根拠があるが第三者検証がない → Medium（10-15点減点）
- 曖昧なビジュアル表現（緑色多用、葉っぱマーク等）と組み合わせ → 追加5-10点減点

### 1.2 オフセットベースの気候中立主張の禁止
**法的根拠**: Directive 2024/825, Annex I, Point 11d

**禁止の核心:**
温室効果ガス排出のオフセットに基づく、製品が環境に対して中立、削減された、またはプラスの影響を持つという主張は、製品自体またはその供給・生産に関するものであると消費者に誤解させ、または製品の消費が環境に影響を与えないという誤った印象を与えるため禁止される。

**禁止される主張の具体例（原文）:**
- "climate neutral"
- "CO2 neutral certified"  
- "carbon positive"
- "climate net zero"
- "climate compensated"
- "reduced climate impact"
- "limited CO2 footprint"
- "100% CO2 compensated"

**日本語表現：**
- 「カーボンオフセットにより気候中立」
- 「カーボンクレジット購入で実質ゼロ」
- 「CO2排出権取引により脱炭素」
- 「植林活動によりカーボンニュートラル」
- 「オフセットで実質排出ゼロ」
- 「カーボンクレジットで気候への影響ゼロ」
- 「J-クレジットでCO2ゼロ」

**重要な原則:**
こうした主張は、**製品の実際のライフサイクル影響**に基づく場合のみ許可され、**製品のバリューチェーン外での温室効果ガスオフセットに基づいてはならない**。前者と後者は等価ではない。

**環境イニシアチブへの投資の広告は可能:**
企業がカーボンクレジットプロジェクトを含む環境イニシアチブへの投資を広告することは禁止されない。ただし、以下の条件を満たす必要がある:
- 誤解を招かない方法で提供
- EU法の要件に準拠
- オフセットと実際の排出削減を明確に区別

**グリーンクレーム指令提案の追加要件:**
（提案版だが、実務上の参考として）
- オフセットは温室効果ガス排出と**別個に報告**すること
- オフセットが**排出削減か除去か**を明記
- オフセットの**高い完全性と正確な会計処理**を確保
- オフセットに依存する割合を明示

**診断ポイント（High risk）：**
- オフセットのみに依存した気候中立主張 → High（25-30点減点）
- 自社排出削減とオフセットの区別不明確 → High（20-25点減点）
- オフセットの詳細情報なし（種類、認証、割合等） → 追加10-15点減点
- オフセットと製品ライフサイクル排出の混同 → 追加10点減点

### 1.3 全体主張の禁止（一部のみの特性を全体として主張）
**法的根拠**: Directive 2024/825, Annex I, Point 11c

**禁止される主張:**
製品の特定の側面または事業者の事業活動の特定の代表的ではない部分にのみ関するものであるにもかかわらず、製品全体または事業者の事業全体について環境主張を行うこと。

**具体例（指令より）:**
- ❌「リサイクル素材製」→ 実際はパッケージのみがリサイクル素材
- ❌「再生可能エネルギーのみ使用」→ 実際は複数の事業施設がまだ化石燃料使用

**追加の重要な指摘:**
この禁止は、事業全体について環境主張を行うことを妨げるものではない。ただし、以下の条件を満たす必要がある:
- 主張が正確で検証可能
- 環境利益を誇張しない
- 例：化石燃料使用の減少を全体として報告する場合、第二の例のような誤解を招く主張にならない

**日本語表現：**
- 「リサイクル素材使用」（実際は包装のみ）
- 「再生可能エネルギー100%」（実際は一部施設のみ）
- 「カーボンフットプリント削減」（実際は製造工程の一部のみ）
- 「持続可能な原材料」（実際は一部原料のみ）
- 「低炭素製品」（実際は製品の一部工程のみ）

**許容される条件：**
- 主張の範囲を明確に限定（例：「パッケージにリサイクル素材使用」）
- 全体に占める割合を明示（例：「原材料の30%が再生可能」）
- 対象範囲を具体的に示す（例：「国内3工場で再生エネルギー使用」）

**診断ポイント（High risk）：**
- 一部特性を製品・企業全体のように主張 → High（20-25点減点）
- 範囲の限定が曖昧・不明確 → Medium（10-15点減点）
- 具体的割合・範囲の記載なし → 追加5-10点減点

### 1.4 未来の環境目標の主張（検証可能なコミットメントなし）
**法的根拠**: Directive 2024/825, Article 6(2) 追加

**禁止の対象:**
環境主張、特に気候関連主張は、特定の日付までに炭素または気候中立、または類似の目標への移行という形で、将来のパフォーマンスに関連することが増えている。こうした主張を通じて、事業者は消費者に対し、製品を購入することで低炭素経済に貢献しているという印象を与える。

**必須要件（すべて満たす必要あり）:**
1. **明確で客観的な公開コミットメント**
2. **検証可能な目標**
3. **詳細で現実的な実施計画**（以下を含む）:
   - コミットメントと目標をどのように達成するかを示す
   - そのための資源を配分
   - 予算資源
   - 技術開発（適切かつEU法に従って）
   - マイルストーン（中間目標）
4. **第三者専門家による検証:**
   - 事業者から独立
   - 利益相反なし
   - 環境問題の経験と能力
   - コミットメントと目標の進捗を定期的に監視
5. **第三者専門家の定期的な調査結果を消費者が入手可能**

**具体例:**
- ❌「2030年までにカーボンニュートラル達成予定」（具体的計画なし）
- ❌「2050年までに脱炭素社会実現」（測定可能な目標なし）
- ❌「将来的に100%再生可能エネルギーに移行」（タイムラインなし）

**日本語表現:**
- 「○○年までにカーボンニュートラル」
- 「○○年までにネットゼロ」
- 「将来的にCO2排出ゼロ」
- 「○○年までに脱炭素」
- 「段階的に排出削減」
- 「環境負荷ゼロを目指します」
- 「2050年カーボンニュートラルに向けて」

**診断ポイント（High risk）：**
- 5要素のうち0-2要素のみ → High（25-30点減点）
- 3要素のみ → Medium（15-20点減点）
- 4要素のみ → Low（10点減点）
- 独立第三者検証なし → 追加10-15点減点
- 進捗の定期公開なし → 追加5-10点減点

---

### 1.5 未認証の持続可能性ラベル・マークの禁止
**法的根拠**: Directive 2024/825, Annex I, Point 11a

**禁止される表現:**
認証スキームに基づかない、または公的機関によって確立されていない持続可能性ラベルを表示すること。

**認証スキームの最低条件:**
持続可能性ラベルを表示する前に、事業者は以下を確認する必要がある:
- 公開されている認証スキームの条件により、透明性と信頼性の最低条件を満たす
- **スキーム要件への準拠の客観的な監視**が存在
- 監視は第三者が実施
- 第三者の能力と独立性が保証（国際・EU・国内基準に基づく）
  - 例：ISO 17065準拠
  - Regulation (EC) 765/2008のメカニズム

**具体例:**
- ❌ 自社独自の「エコマーク」「環境認証マーク」（第三者検証なし）
- ❌ 検証不可能な「カーボンニュートラル認証」
- ❌ 基準が不明確な「気候フレンドリー認定」

**日本語での該当表現:**
- 自社独自の「環境配慮マーク」
- 検証不可能な「エコ認証」「グリーン認定」
- 基準不明の「低炭素認証」
- 「当社基準クリア」マーク
- 「自社認定」バッジ

**許容される認証:**
- 公的機関（政府、自治体等）による認証
- EU Ecolabel (Regulation 66/2010)
- EMAS logos (Regulation 1221/2009)
- 有機農産物ロゴ
- 透明性・公平性・独立性のある第三者認証
- 認証基準が公開されている
- 違反時の措置が明確

**診断ポイント（High risk）：**
- 自己作成ラベルの使用（第三者検証なし） → High（20-30点減点）
- 認証機関が不明確・独立性に疑問 → High（15-20点減点）
- 認証基準が非公開 → 追加10点減点
- 監視・違反措置の仕組みなし → 追加5-10点減点

---

### 1.6 法的義務を特別な取り組みとして主張することの禁止
**法的根拠**: Directive 2024/825, Annex I, Point 23b

**禁止される主張:**
EU市場における関連製品カテゴリー内のすべての製品（輸入製品を含む）に法律によって課されている要件を、事業者の提供の特徴的な特性として提示すること。

**具体例（指令より）:**
- ❌ ある製品カテゴリーのすべての製品に対してEUで法律により禁止されている特定の化学物質を含まないことを広告する

**例外:**
この禁止は、以下の商業慣行には適用されない:
- ✅ 一部の製品にのみ適用される法的要件への準拠を宣伝する場合
- ✅ 同じカテゴリーの他の競合製品には適用されない要件（例：非EU原産製品）

**例（指令より）:**
EU法に従って持続可能に生産された魚製品について、第三国原産の魚製品がそのEU法的要件に準拠する必要がない場合、持続可能性特性を宣伝することは許可される。

**日本語表現:**
- 「法令基準をクリア」を「環境性能が高い」と表現
- 「規制遵守」を「業界最先端の取り組み」と表現
- 「省エネ法対応」を「エコ製品」と表現
- 「フロン規制対応」を「環境配慮型」と表現

**診断ポイント（Medium risk）：**
- 法定義務を特別な取り組みとして強調表現 → Medium（15-20点減点）
- 法定基準超過分が不明確 → 追加10点減点
- すべての競合製品に適用される義務なのに特徴として主張 → High（20点減点）

---

### 1.7 比較環境主張の透明性要件
**法的根拠**: Directive 2024/825, Article 7(3)

**必須の情報開示:**
製品の環境的または社会的特性や循環性側面（耐久性、修理可能性、リサイクル可能性等）に基づく比較を行う場合、事業者は以下を消費者に提供する必要がある:
- **比較の方法**
- **比較の対象となる製品**
- **それらの製品の供給者**
- **情報を最新に保つための措置**

**客観性の確保:**
- 同じ機能を果たす製品を比較
- 共通の方法と共通の仮定を使用
- 比較される製品の重要で検証可能な特徴を比較

**禁止される主張（Annex I追加）:**
- ❌ 紙製品に「プラスチックを含まない」（無関係な利点を強調し、実際には持たない比較優位を示唆）
- ❌ 市場に存在しない製品との比較

**日本語表現:**
- 「従来品より50%削減」（従来品の特定なし、基準年不明）
- 「競合他社より環境性能が優れている」（比較方法不明）
- 「業界平均以下の排出量」（業界平均の算出根拠なし）
- 「当社比○○%削減」（比較基準年・製品不明）

**診断ポイント（Medium-High risk）：**
- 比較対象・方法が完全に不明 → High（20-25点減点）
- 比較対象は明示されているが方法が不明 → Medium（10-15点減点）
- 比較時点が古い（3年以上前） → 追加5点減点
- 市場に存在しない製品との比較 → High（20点減点）

### 1.8-1.13 早期陳腐化関連の禁止事項
（ClimateWash特化版では優先度低いため、簡略記載）

**1.8**: ソフトウェア更新が機能に悪影響を与える情報の非開示
**1.9**: 機能更新を適合性維持に必要と虚偽表示
**1.10**: 耐久性制限機能を含む製品の宣伝
**1.11**: 耐久性に関する虚偽主張

## 2. EUグリーンクレーム指令（提案版）による実証要件

※2025年6月に撤回表明されたが、実務上のベストプラクティスとして参照価値あり

### 2.1 科学的根拠の要件
**法的根拠**: COM(2023) 166, Article 3(1)

**必須要素:**
明示的環境主張を実証する評価は以下を満たす必要がある:

a) **主張の範囲の明示**
   - 製品全体か一部か
   - ライフサイクル全体か特定段階か
   - 事業者の全活動か一部か

b) **広く認められた科学的証拠に依拠**
   - 正確な情報を使用
   - 関連する国際基準を考慮

c) **ライフサイクル視点での重要性の実証**
   - 主張の対象となる環境影響・側面・性能がライフサイクル視点で重要であることを実証

d) **包括的な評価**
   - 環境性能評価に重要なすべての環境側面・影響を考慮

e) **法的要件との非等価性**
   - 主張が製品グループ内の製品または部門内の事業者に課される法的要件と等価でないことを実証

f) **一般的慣行との比較**
   - 製品または事業者が、関連製品グループまたは部門の一般的慣行と比較して、環境影響・側面・性能において著しく優れているかの情報提供

g) **トレードオフの特定**
   - 主張の対象となる環境影響・側面・性能の改善が、以下に関して著しい害をもたらすかを特定:
     - 気候変動
     - 資源消費と循環性
     - 水資源と海洋資源の持続可能な利用と保護
     - 汚染
     - 生物多様性
     - 動物福祉
     - 生態系

h) **GHGオフセットの分離報告**
   - 使用されたGHGオフセットをGHG排出と分離
   - オフセットが排出削減か除去かを明記
   - オフセットの高い完全性と正確な会計処理を説明

i) **一次情報の使用**
   - 事業者が入手可能な一次情報を含める
   - 主張の対象となる環境影響・側面・性能に関して

j) **二次情報の使用**
   - 一次情報が入手できない場合、代表的な二次情報を含める

**診断ポイント:**
- 科学的根拠の文献引用なし → Medium（10-15点減点）
- ライフサイクル評価（LCA）なし → High（20-25点減点）
- トレードオフの評価なし → High（15-20点減点）
- 一般的慣行との比較なし → Medium（10-15点減点）

---

### 2.2 気候関連主張の特別要件
**法的根拠**: COM(2023) 166, Recital 21

**オフセットに基づく気候関連主張の課題:**
気候関連主張は、特に不明確で曖昧であり、消費者を誤解させる傾向がある。これは特に、「気候中立」「カーボンニュートラル」「100% CO2補償」または特定の年までに「ネットゼロ」になる、または類似の環境主張に関連する。

このような主張は、企業のバリューチェーン外の「炭素クレジット」を通じた温室効果ガス排出の「オフセット」、例えば森林または再生可能エネルギープロジェクトからのクレジットに基づくことが多い。オフセットの基礎となる方法論は大きく異なり、常に透明で正確、一貫しているわけではない。

**主なリスク:**
- 過大評価
- 二重計上
- 追加性の欠如
- 永続性の欠如
- 野心的で動的なベースラインの欠如
- 正確な会計処理の欠如

**優先すべきアプローチ:**
全球的な気候変動緩和目標に適切に貢献するために、事業者は自社の事業とバリューチェーン全体にわたる排出の効果的な削減を優先すべきであり、オフセットに依存するのではなく。

**残余排出への対処:**
結果として生じる残余排出は、セクター別の経路によって異なり、グローバルな気候目標に沿って、除去の強化を通じて対処する必要がある。

**オフセット使用時の透明性要件:**
- 事業者または製品のGHG排出と**分離して**報告
- **総排出量のうちオフセットで対処される割合**を明記
- オフセットが**排出削減か除去強化か**を明記
- **適用された方法論**を説明

**診断ポイント（High risk）：**
- オフセット情報の分離報告なし → High（20-25点減点）
- オフセットの種類（削減/除去）不明 → High（15-20点減点）
- オフセット割合不明 → Medium（10-15点減点）
- 自社バリューチェーン内の削減が不明確 → High（20点減点）
- 残余排出の定義なし → Medium（10点減点）

---

### 2.3 比較主張の追加要件
**法的根拠**: COM(2023) 166, Article 4

比較環境主張（他製品・事業者より環境影響が少ない/多い、または環境性能が良い/悪いという主張）は、Article 3に加えて以下を満たす必要がある:

a) **同等の情報・データ使用**
b) **同等の方法でのデータ生成・調達**
c) **バリューチェーンの段階カバレッジが同等**（最重要段階を含む）
d) **環境影響・側面・性能のカバレッジが同等**（最重要なものを含む）
e) **比較に使用する仮定が製品・事業者間で一致**

**改善主張の追加要件（Article 4(2)）:**
同一事業者の旧製品または市場にない競合事業者の製品との比較の場合:
- 改善が他の環境影響・側面・性能にどう影響するか説明
- **比較のベースライン年を明記**

**診断ポイント（High risk）：**
- データ・方法が不同等 → High（20点減点）
- カバレッジが不同等 → High（15-20点減点）
- ベースライン年不明（改善主張） → Medium（10-15点減点）

---

### 2.4 第三者検証の要件
**法的根拠**: COM(2023) 166, Article 10, 11

**検証の義務:**
明示的環境主張の実証とコミュニケーションは、主張が商業コミュニケーションで使用される**前に**第三者によって検証され、認証される必要がある。

**検証者の要件:**
- 公式に認定された機関
- **独立した組織**
- **利益相反なし**
- 最高度の専門的誠実性
- 必要な専門知識、設備、インフラ
- 適切な人員配置
- **職業上の秘密を守る**

**適合証明書（Certificate of Conformity）:**
- 検証後に発行
- **EU全域で認識**
- 加盟国間で情報共有（IMI System経由）
- 内部市場全域での主張使用を可能に

**診断ポイント（High risk）：**
- 第三者検証なし → High（25-30点減点）
- 検証者の独立性不明・利益相反の疑い → High（20-25点減点）
- 検証証明書の提示なし → High（15-20点減点）
- 検証者の認定情報不明 → Medium（10-15点減点）

---

### 2.5 情報開示・コミュニケーション要件
**法的根拠**: COM(2023) 166, Article 5

**開示必須情報（主張と共に提供）:**
以下の情報を物理形式、またはWeblink/QRコード等で提供:

a) 主張の対象となる**環境側面・影響・性能**
b) 関連する**EU・国際基準**（該当する場合）
c) **基礎となる研究・計算**
   - 結果を省略しない
   - 範囲・仮定・限界を説明
   - ただし営業秘密（Directive 2016/943）は除く
d) 主張の対象となる**改善の達成方法**の簡単な説明
e) **適合証明書**と**検証者の連絡先**
f) **気候関連主張でオフセット使用の場合**:
   - オフセットに依存する程度
   - 排出削減か除去かの情報
g) **消費者向けの明確で理解しやすい評価要約**
   - 主張が行われる加盟国の公用語で少なくとも1つ提供

**追加要件:**
- 最終製品で使用段階が最も重要なライフサイクル段階の場合、**消費者の適切な使用方法に関する情報**を含める
- 将来の環境性能に関する主張の場合、**自社事業とバリューチェーン内の改善への時限付きコミットメント**を含める

**診断ポイント:**
- 7要素中0-3要素のみ開示 → High（20-25点減点）
- 4-5要素のみ開示 → Medium（10-15点減点）
- 6要素のみ開示 → Low（5点減点）
- 消費者向け要約なし → 追加10点減点
- 日本語での情報提供なし（日本市場） → 追加5点減点

## 3. 日本語表現特有の曖昧さによるClimateWash

### 3.1 曖昧な削減表現

**問題となる表現：**
- 「削減に取り組んでいます」（結果・程度不明）
- 「削減を目指します」「削減を目指しています」（目標値・期限不明）
- 「削減に努めます」「削減に努めています」（具体的行動不明）
- 「削減に配慮しています」（何をしているか不明）
- 「削減に向けた活動」（具体性なし）
- 「削減につながる」（因果関係不明確）

**診断ポイント：**
- 具体的数値なし → Medium（10-15点減点）
- 基準年・期限なし → 追加5-10点減点
- 削減の対象範囲不明 → 追加5点減点

---

### 3.2 曖昧な主語・責任主体

**問題となる表現：**
- 主語が不明確で誰が行動するか不明
- 責任主体が曖昧
- 「みんなで脱炭素」「社会全体で取り組む」（責任分散）
- 「〜が期待されます」「〜とされています」（誰がそう言っているのか不明）

**診断ポイント：**
- 責任主体不明 → Medium（10-15点減点）
- 自社の役割・貢献が不明確 → 追加5-10点減点

---

### 3.3 ポジティブワードの多用

**問題となる表現：**
- 「やさしい」「想い」「未来のために」「地球のために」等の情緒的表現
- 具体的行動を伴わないスローガン
- イメージ優先の表現
- 「心がける」「大切にする」等の精神論

**診断ポイント：**
- 情緒的表現のみで実質的内容なし → High（15-20点減点）
- スローガンに具体的裏付けなし → Medium（10-15点減点）

---

### 3.4 受動態・婉曲表現

**問題となる表現：**
- 「考えられます」「期待されます」「見込まれます」
- 「可能性があります」「〜かもしれません」
- 「〜とされています」「〜と言われています」
- 「一定の効果が」「ある程度の削減」

**診断ポイント：**
- 断定を避ける表現の多用 → Medium（10点減点）
- 定量化可能なのに曖昧表現 → 追加5点減点

---

### 3.5 「〜フリー」表現の注意点

**問題となる表現：**
- 「カーボンフリー」「CO2フリー」
- 「排出フリー」

これらは「ゼロ」を意味するが、実際はオフセットに基づく場合が多い → 1.2の違反に該当する可能性が高い。

**診断ポイント：**
- オフセットベースの「フリー」表現 → High（20-25点減点）

---

## 4. ビジュアル要素によるClimateWash

### 4.1 色彩による印象操作

**問題となる要素：**
- 緑色の過度な使用（環境=緑の連想）
- 青色（水・空）の多用（クリーンなイメージ）
- 白・ライトトーン（クリーンさ・純粋さの演出）

**診断ポイント：**
- 環境イメージ色の過度使用（画面・パッケージの50%以上） → Medium（5-10点減点）
- 実際の環境性能と色彩イメージの乖離 → 追加5点減点

---

### 4.2 自然イメージの使用

**問題となる要素：**
- 葉っぱ、樹木、森林のイメージ（CO2吸収源の象徴）
- 地球・惑星のイメージ（地球保護の象徴）
- 動物・生態系のイメージ（生物多様性の象徴）
- 水・海のイメージ（クリーンな資源）
- 空・雲のイメージ（大気の清浄さ）
- 風車・太陽光パネルのイメージ（実際は使用していない）

**重要:** 指令Recital 9では、「色彩や画像と組み合わされた主張も一般的環境主張を構成しうる」と明記。

**診断ポイント：**
- 自然イメージと製品の関連性なし → High（15-20点減点）
- 自然イメージと実際の環境性能の乖離大 → High（20-25点減点）
- 再生エネルギー画像使用（実際は不使用） → High（20点減点）

---

### 4.3 シンボル・アイコンの使用

**問題となる要素：**
- リサイクルマーク（実際にリサイクル不可）
- ハートマーク（愛・配慮の象徴、根拠なし）
- 太陽・風車（再生エネルギー非使用なのに表示）
- 手と地球（保護のイメージ、根拠なし）
- 自己作成の「エコマーク」（1.5の違反）

**診断ポイント：**
- ミスリーディングなシンボル使用 → High（15-20点減点）
- 未認証マークの使用 → High（20-30点減点）

---

### 4.4 製品と無関係な自然風景

**問題となる要素：**
- 製品の製造・使用と関係ない美しい自然風景
- 化石燃料製品なのに再生可能エネルギー施設の画像
- 高排出製品なのに森林・海洋の画像
- 航空機の広告に森林の画像
- 自動車の広告に氷河・北極の画像

**診断ポイント：**
- 製品と画像の関連性なし → High（20-25点減点）
- 逆印象（高排出+クリーンイメージ） → 追加10点減点

---

## 5. 総合評価基準

### 点数化の方法

**基本点：100点**

各違反項目について減点し、最終スコアを算出

**最終スコア = 100点 - 各違反項目の減点合計**

### 評価レベル

**High Risk（最終スコア0-15点、減点85-100点）：重大なClimateWash**
- 法的リスク極めて高い
- 即座に是正が必要
- 訴訟・罰金のリスク
- 例：
  - オフセットのみのカーボンニュートラル主張
  - 第三者検証なしの一般的環境主張
  - 未認証の自社ラベル使用
  - 科学的根拠なしの気候中立主張

**Medium Risk（最終スコア16-50点、減点50-84点）：中程度のClimateWash**
- 法的リスクあり
- 早期是正推奨
- レピュテーションリスク
- 例：
  - 根拠不十分な削減目標
  - 範囲不明確な環境主張
  - オフセット情報の不透明性
  - 比較方法の不明確さ

**Low Risk（最終スコア51-85点、減点15-49点）：軽微なClimateWash**
- 改善の余地あり
- 情報追加で是正可能
- 予防的対応推奨
- 例：
  - 比較時点が古い
  - 情報開示が部分的
  - 一部情報のアクセス困難

**Compliant（最終スコア86-100点、減点0-14点）：適切な表現**
- 法的リスク低い
- 軽微な改善のみ推奨
- ベストプラクティス

**Before（NG）:** 「2030年脱炭素目指します」

- 「Scope 3排出量については、サプライヤーからのデータ取得が困難なため、業界平均値を使用」

## 10. 補足：ClimateWash判断における注意点

### 10.1 文脈の重要性
同じ表現でも、文脈や付随情報によって判断が変わる。例：
- 「カーボンニュートラル」単独 → 違反
- 「カーボンニュートラル」+詳細な実証情報+第三者検証 → 可能性あり（ただし慎重に）

### 10.2 業界基準の考慮
業界ごとに技術的制約や標準的表現が異なる:
- 航空業界：SAF（持続可能な航空燃料）使用率
- 建築業界：ライフサイクルカーボン（WBLCA）
- 食品業界：農業排出の測定課題

### 10.3 文化的要因
日本特有の婉曲表現文化を考慮しつつ、EU基準への準拠を目指す:
- 「努力する」「配慮する」→ 具体的コミットメントへ
- 「目指す」→ 時限付き目標へ

### 10.4 進化する基準
科学的知見と法規制の進展に応じて基準も更新が必要:
- 気候科学の最新知見（IPCC報告書等）
- カーボンクレジットの質基準の進化
- LCA方法論の改善

### 10.5 意図と結果
悪意はなくても結果的にミスリーディングなら該当:
- 善意の誇張 → 違反
- 不完全な理解に基づく主張 → 違反
- 「知らなかった」は抗弁にならない
